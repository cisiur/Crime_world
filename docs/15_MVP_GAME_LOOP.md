# MVP Game Loop

## Purpose

This document describes the first playable experience of the MVP.

The goal is to make sure the player understands the fantasy, starts making decisions quickly, and feels progress within the first session.

## First 5 minutes

The player:
- creates a criminal character,
- picks basic traits,
- enters a single playable city,
- starts with one tiny hideout or safe location,
- sees the first district overview,
- gets the first small opportunity.

## First 15 minutes

The player should begin to:
- recruit a first helper,
- perform a low-risk operation,
- gain first income,
- see the first reaction from the world,
- learn that actions create pressure.

## First 30 minutes

The player should be able to:
- repeat small operations,
- expand the crew,
- buy or take over a basic business,
- learn which district is safer or more profitable,
- feel the first strategic tradeoff between money and attention.

## First session arc

The first session should ideally move through:
1. survival,
2. first profit,
3. first pressure,
4. first expansion,
5. first real decision about risk.

## Core emotional beats

The player should feel:
- weak at the start,
- clever after the first successful operation,
- hungry after the first income spike,
- worried when pressure rises,
- ambitious when expansion becomes possible.

## MVP pacing principle

The MVP should not wait too long to show consequence.
The player must see that the city responds, rivals react, and growth changes the game.

## Success condition for the MVP

The MVP works if the player wants to start a second run after the first one ends or stabilizes.
